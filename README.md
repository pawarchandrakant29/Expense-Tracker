Expense Tracker is a modern web application developed using Node.js and Express.js for the backend, and MongoDB for the database. This project provides a user-friendly interface to manage income and expenses, categorize transactions, and generate detailed reports.
<br>
<h3>Features</h3>
<ul>
  <li>
      Expense and Income Management: Add, edit, and delete income and expense entries with detailed descriptions, amounts, and categories.
  </li>
  <li>
      Categorization: Organize transactions into categories for easier tracking and reporting.
  </li>
  <li>
      Total Saving Reports: View comprehensive reports of all total income , expense & savings .
  </li>

</ul>
<h3>Technology Stack</h3>
<ul>
  <li>
      Frontend: EJS (Embedded JavaScript templates), HTML5, CSS3, Bootstrap (optional for styling)
  </li>
  <li>
      Backend: Node.js, Express.js
  </li>
  <li>
      Database: MongoDB
  </li>
  <li>
      Version Control: Git, GitHub
  </li>
  
</ul>
